LocatoBot (BaseHacks Hackathon - Top Ten Winner, Best Location-Based Hack, Best Conversation/Interaction Bot)

LocatoBot averages the location of all participants in a Telegram group chat, and then finds restaurants, etc. in that general location. It is intended to help faciltate meetups.

LocatoBot was programmed in JavaScript, using the Google Places API as well as the GupShup bot platform.
